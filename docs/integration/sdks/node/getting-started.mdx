# Getting Started

This section describes how to use the Botanalytics Node.js SDK for your project.

<V3Warning />

:::info 

For Samsung Bixy, you can skip this step and go directly to the [Samsung Bixby page](samsung-bixby).

:::


## Prerequisites

* An NPM project. For information on how to set up a NPM project, please see [this](https://docs.npmjs.com/getting-started/creating-node-modules).
* A Node.js environment with Node version 6.x or above.

## Installing SDK module

Within your project, run the following command depending on the package manager you use:

<Tabs>
<TabItem value="npm" label="NPM">

```bash
npm install @botanalytics/core
```

</TabItem>
<TabItem value="yarn" label="Yarn">

```bash
yarn add @botanalytics/core
```

</TabItem>
</Tabs>
