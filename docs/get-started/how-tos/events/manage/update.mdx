# Update an Event

## Navigate to Events Dashboard

- From the account dashboard, locate the "Events" section under the "Behavior" tab.
- Click on "Events" to access the events dashboard.
  <img src={EventDashboard} alt="Event Dashboard" />

## Access Event Edit Options

- On the "Events" page, find the event you want to edit.
- In the top right corner, click on the ellipsis button (three dots).
  <img src={EventEllipsis} alt="Event Ellipsis" />

## Open the Dropdown Menu

- In the dropdown menu, click on "Edit."
  <img src={EventClickEdit} alt="Event Click Edit" />

## Navigate to the Event Edit Page

- You will be directed to a new page for editing the selected event.
  <img src={EventUpdatePage} alt="Event Update Page" />

## Edit Event

- On the event edit page, make the necessary changes. You can edit the event name and add conditions.
  - Click on the event name to edit it.
    <img src={EventClicktoUpdateName} alt="Event Click to Update Name" />
  - After clicking, freely edit the name.
    <img src={EventUpdatedName} alt="Event Update Name" />
  - Add a condition by clicking on the "Add Condition" button.
    <img src={EventAddCondition} alt="Event Add Condition" />
  - Name the expression by clicking on the name tile.
    <img src={EventUpdateExpression} alt="Event Update Expression" />

## Confirm Action

- Click on the "Update" button to complete the event editing.
  <img src={EventClickonUpdate} alt="Event Click on Update" />
- A pop-up will appear, confirming the action. Click on "Update" to proceed.
<img src={EventConfirmUpdate} alt="Event Confirm Update" />

## View Updated Event

- After confirming, view your updated event on the Events Dashboard.
  <img src={EventUpdatedView} alt="Event Updated View" />
