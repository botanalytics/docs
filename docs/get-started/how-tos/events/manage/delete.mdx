# Delete an Event

## Navigate to Events Dashboard

- From the account dashboard, locate the "Events" section under the "Behavior" tab.
- Click on "Events" to access the events dashboard.
  <img src={EventDashboard} alt="Event Dashboard" />

## Find the Event to Remove

- Identify the event you wish to remove from the list on the "Events" page.
- Next to the event, find the ellipsis button (three dots).
  <img src={EventEllipsis} alt="Event Ellipsis" />

## Open the Dropdown Menu

- Click on the ellipsis button to open a dropdown menu with various options.
  <img src={EventDropdown} alt="Event Dropdown" />

## Select "Remove" to Remove the Event

- From the dropdown menu, choose the "Remove" option. This initiates the process of removing the event.
  <img src={EventClickRemove} alt="Event Click Remove" />

## Confirm Action 

- A confirmation prompt will ask, "Are you sure you want to delete the event?"
- Click on "Remove" to confirm and complete the process of removing the event.
  <img src={EventConfirmDelete} alt="Event Confirm Delete" />

## Event Removed From Dashboard 

- By following these steps, successfully remove the event from Botanalytics.
- Please be certain about this action, as it cannot be undone.
  <img src={EventDeleteView} alt="Event Delete View" />
