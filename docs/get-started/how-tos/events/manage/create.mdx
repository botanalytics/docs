# Create an Event

## Navigate to Events Dashboard
- From the account dashboard, locate the "Events" section under the "Behavior" tab.
- Click on "Events" to access the events dashboard.
  <img src={EventDashboard} alt="Event Dashboard" />

## Locate Add Event Button

- At the top right corner of the events dashboard, you'll find an "Add Event" button. Click on it.
  <img src={EventAddEventButton} alt="Event Add Event Button" />

## Enter Event Details

- A pop-up window will appear. In this window, provide a descriptive name for your new event.
  <img src={EventNameTile} alt="Event Name Tile" />

- Name the associated expression.
  <img src={EventNameExpression} alt="Event Name Expression" />

## Add Event

- After naming and configuring the event, click the "Add" button.
  <img src={EventClickonAdd} alt="Event Click on Add" />

## View Created Event

- You will be able to view your newly added event in the Events dashboard.
  <img src={EventView} alt="Event View" />
