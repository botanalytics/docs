# Rename a Project

## There are two methods for renaming a project:

## Method 1: Renaming a Project from the Dropdown Menu

### Access Dropdown Menu 

   - In your account dashboard, locate the name of your active project.
   - Click on the project name to access the dropdown menu.
   <img src={DashboardActiveProject} alt="Dashboard Active Project" />

### Access Project Settings

   
   - From the dropdown menu, click on "Project settings" to access the project settings page.
   <img src={DashboardProjectSettings} alt="Dashboard Project Settings" />

### Navigate to the "General" Tab

   - In the "Project Settings" page, navigate to the "General" section.
   - Under this section, find the "Name" tab with the current project name displayed.
   <img src={RenameClickOnTile} alt="Rename Click on Tile" />

### Edit the Project Name

   - Click on the current project name to edit it to the desired name.
   <img src={RenameClickRenameInput} alt="Rename Click Rename Input" />

### Click on "Rename"

   - After editing the project name, click on the "Rename" button.
   <img src={RenameRenameButton} alt="Rename Rename Button" />

### Project Renamed

   Your project will immediately be renamed to the new name you specified.
   <img src={RenameViewRenamed} alt="Rename View Renamed" />

## Method 2: Renaming a Project from the Projects Dashboard

### Go to the Projects 

   - Click on the Botanalytics logo in the top left corner to access the projects dashboard.
   <img src={ProjectClickLogo} alt="Project Click Logo" />

### Locate the Project to Rename

   - In the projects dashboard, locate the project you wish to rename.
   <img src={ProjectViewTile} alt="Projects View Tile" />

### Access Project Settings

   - Click on the ellipsis icon (three dots) in the top right corner of the project tile.
   - From the dropdown menu, click on "Settings" to access the Project Settings page.
   <img src={ProjectClickEllipsis} alt="Project Click Ellipsis" />
   <img src={ProjectClickSettings} alt="Rename Project Click Settings" />

### Navigate to the "General" Tab

   - In the "Project Settings" page, navigate to the "General" section.
   - Under this section, find the "Name" tab with the current project name displayed.
   <img src={RenameClickOnTile} alt="Rename Click on Tile" />

### Edit the Project Name

   - Click on the current project name to edit it to the desired name.
   <img src={RenameClickRenameInput} alt="Rename Click Rename Input" />

### Click on "Rename"

   - After editing the project name, click on the "Rename" button.
   <img src={RenameRenameButton} alt="Rename Rename Button" />

### Project Renamed

   Your project will immediately be renamed to the new name you specified.
   <img src={RenameViewRenamed} alt="Rename View Renamed" />
