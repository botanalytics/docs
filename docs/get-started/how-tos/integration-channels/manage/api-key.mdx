# Re-generate API Key for an Integration Channel

## There are two methods for regenerating API key for a channel:

## Method 1: Regenerating from the accounts dashboard

### Access Dropdown Menu 

- In your account dashboard, locate the name of your active project.
- Click on the project name to access the dropdown menu.
  <img src={DashboardActiveProject} alt="Dashboard Active Project" />

### Access Project Settings
- On the dropdown menu click on Project settings to take you to the project settings page.
  <img src={DashboardClickProjectSettings} alt="Dashboard Click Project Settings" />

### Navigate to Channels
- In the settings window, you will see different sections, including "General" and "Channels."
- Click on "Channels" to manage your connected channels.
  <img src={ChannelsChannelsTab} alt="ChannelsChannelsTab" />

### Select the Channel to Regenerate API Key
- Click on the specific Integration channel for which you want to regenerate the API key. This will open the channel's settings.
  <img src={RegenClickIntChannel} alt="Regen Click Int Channel" />

### Regenerate the API Key
- In the channel's settings, look for the "Regenerate" button on the right side. Click on this button to regenerate the API key for the selected Integration channel.
  <img src={RegenIntClickRegen} alt="Regen Int Click Regen" />

### Confirm Regeneration
- A prompt will appear, asking you to confirm if you want to regenerate the API key. Click on "Regenerate" to proceed.
  <img src={RegenIntConfirm} alt="Regen Int Confirm" />

### New API Key
- After confirming, the API key for the selected Integration channel will be regenerated, and a new unique API key will be generated for that Integration channel.
  <img src={RegenIntView} alt="Regen Int View" />


## Method 2: Regenerating from the Projects Dashboard

### Access the Dashboard
- Click on the Botanalytics logo in the top left corner to access the Projects dashboard.
  <img src={ProjectClickLogo} alt="Project Click Logo" />

### Access Project Settings
- In the Projects dashboard, locate the project you want to add a new Integration channel to.
  <img src={ProjectViewTile} alt="Projects View Tile" />

### Navigate to Channels
- In the settings window, you will see different sections, including "General" and "Channels."
- Click on "Channels" to manage your connected channels.

### Select the Integration Channel to Regenerate API Key

  <img src={RegenIntClickRegen} alt="Regen Int Click Regen" />

### Confirm Regeneration

  <img src={RegenIntConfirm} alt="Regen Int Confirm" />

### New API Key

  <img src={RegenIntView} alt="Regen Int View" />
